SOLO PARA LA PC DE CAJA
=======================

Opcion A — Acceso directo con logo AgilDTE (recomendado)
  1. Copie al escritorio la carpeta scripts\ (o estos archivos):
       crear_acceso_directo_caja.bat
       crear_acceso_directo_caja.ps1
       iniciar_chrome_pos_impresion_silenciosa.bat
       agildte_pos_logo.png   (o agildte_pos.ico si ya existe)
  2. Doble clic en: crear_acceso_directo_caja.bat
  3. En el Escritorio aparece: "AgilDTE - Punto de venta" con el icono del logo.
  4. Use SIEMPRE ese acceso directo (no el .bat suelto).

Opcion B — Solo el .bat (sin icono personalizado)
  AgilDTE_Abrir_Caja.bat

NO necesita copiar el proyecto, Python ni Docker.
El POS esta en: https://agildte.com/pos/ventas_pos

En la caja:
  - Google Chrome instalado
  - Internet
  - Impresora EPSON como PREDETERMINADA en Windows

Uso diario: doble clic en el acceso directo "AgilDTE - Punto de venta"
Cerrar: Alt+F4
